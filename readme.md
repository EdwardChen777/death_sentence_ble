## Setup

1. **Clone the repository**:

```bash
git clone <repository_url>
cd deathscent_ui
``` 

2. **Creating virtual environment**:
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
``` 

3. **Running main.py**:
```bash
cd testing
python main.py
```

4. **Open up frontend html file and run backend**:
```bash
python run.py
```
